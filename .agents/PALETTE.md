# Paleta de Colores Lumi AI

Esta es la paleta de colores oficial de Lumi AI. Toda futura actualización debe respetar estos colores. El viejo color de "KTM Duke" (Naranja #ff6600 y rgba(255, 102, 0)) **ha sido eliminado y prohibido**.

## Colores Principales
- **Azul Cian Principal:** `#00a8ff` (RGB: 0, 168, 255)
- **Azul Cian Oscuro (Hover):** `#0090ff`
- **Cian Brillante (Gradiente 1):** `#00d2ff`
- **Morado/Magenta Brillante (Gradiente 2):** `#a800ff`

## Estilos de Botones y Acentos (Gradients)
Cuando se creen botones o se quieran hacer elementos estéticos ("bonitos" y "Premium"), se deben usar gradientes en lugar de colores sólidos.
- **Gradiente Base:** `linear-gradient(135deg, #00d2ff 0%, #a800ff 100%)`
- **Gradiente Hover:** `linear-gradient(135deg, #00e1ff 0%, #ba28ff 100%)`
- **Sombra (Glassmorphism):** `box-shadow: 0 4px 12px rgba(168, 0, 255, 0.25)`

## Transparencias
Siempre que necesites transparencia, usa la base `0, 168, 255`.
- Ejemplo suave: `rgba(0, 168, 255, 0.1)`
- Ejemplo intenso: `rgba(0, 168, 255, 0.8)`
